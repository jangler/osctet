Osctet

Osctet is a tracker for desktop operating systems. The purpose of the project is to provide a self-contained, natively microtonal music creation tool. The sound engine's design is influenced by older synth/sampler hardware like the Sound Blaster and MT-32, as well as by chipmusic-adjacent software such as pxtone.

Osctet is tested on Windows, Linux, and Wine. macOS builds are provided, but may have platform-specific issues.

For older versions, see the releases page.

Features

  • Alternative tunings
    • Generate octave and non-octave equal temperaments
    • Optimized for medium-to-large equal temperaments
    • Scala scale file support
  • Rhythmic support
    • Flexible beat divisions
    • Rational tempo changes
  • Synthesis engine
    • Use subtractive, additive, or FM synthesis
    • Link parameters via modulation
    • Effects: reverb, delay, compression
  • Sample loading & playback
  • MIDI input (but no recording yet)
  • Multi-track WAV export
  • Customizable, shareable color themes
  • Supported audio hosts: WASAPI, ALSA, JACK, CoreAudio
